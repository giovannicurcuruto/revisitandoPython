# CRUD básico, com exemplo de Flask em Python

* **GET**
> /carros

* **GET by ID**
> /carros/ID

* **POST - Create**
> /carros/ID
> Deve ser adicionado informações no campo BODY

* **DELETE**
> /carros/ID

* **UPDATE**

> /carros/ID
> Deve ser adicionado informações no campo BODY

* **Exemplo de Body:**

>{
>    "ano": 2015,
>    "id": 6,
>    "marca": "Nissan",
>    "modelo": "Versa"
>}
